# グレゴリオ製菓 名刺デザイン再設計案

## 現在の位置づけ

以前の8案比較は、方向性探索としては有効だったが、実用名刺としては装飾過多だった。現在は `デザイン要件.txt` と `DESIGN_SYSTEM.md` を正として、ブラウザ上で各メンバーが編集できる縦長名刺メーカーへ整理している。

## コンセプト

サークル名「グレゴリオ製菓」は、ブルーアーカイブの総力戦ボス「グレゴリオ」が持つ聖歌・オルガン・合唱・聖堂のイメージと、「製菓」の柔らかいユーモアを合わせた名前として扱う。

ただし、名刺上でその背景を説明しすぎない。今回の名刺はサークル紹介画像ではなく、メンバー個人のアカウント情報を読みやすく載せるためのテンプレートである。

## デザイン方針

- 余白を広く取り、情報の優先順位を明確にする。
- 必須情報はサークル名、サークルID、アカウント名、先生名、ロゴに絞る。
- 推し、ひとこと、総力戦ネタ、宣伝文句は入れない。
- ロゴは画像生成候補、追加PNG素材、サブロゴ素材から選べるようにする。
- 主ロゴ、光輪、サブロゴはそれぞれ表示、サイズ、位置を調整できるようにする。
- SVGサブロゴは白地を残し、色付き部分だけをユーザー指定色に変更できるようにする。
- 色は白、アイボリー、淡紫、深紫、焼き菓子ベージュ、金を少量に絞る。

## タイポグラフィ方針

GitHub Pagesで公開し、各メンバーがブラウザで自分の名前を入力する前提にする。そのため、Google FontsのWebフォントを使い、端末にフォントを入れていなくても近い見た目で表示できるようにする。読み込みに失敗した場合は、端末標準の日本語フォントへフォールバックする。

名刺本体は読みやすさを優先し、基本は `Noto Sans JP`、見出しにだけ `Noto Serif JP` を使う。フォントの種類を増やして見せ場を作るのではなく、余白と文字階層で整える。

## 現在の編集機能

- `サンプル縦長` と `標準縦長 55:91` の比率切り替え。
- サークル名、英字コピー、ID、所属表記、先生名の入力。
- 各テキストのフォントと色変更。
- 主ロゴのカテゴリ選択、サイズ、位置調整。
- 主ロゴ用の色変更可能な光輪レイヤー。
- サブロゴの選択、サイズ、位置調整。
- SVGサブロゴの色変更。
- PNG/SVG保存。

## 名刺に載せる情報

- サークル名: グレゴリオ製菓
- サークルID
- アカウント名 / 先生名
- サークルロゴ

## 公募での説明文案

名刺はサークル紹介画像ではなく、各メンバーが使う個人用テンプレートとして作っています。情報はサークル名、サークルID、所属表記、先生名、ロゴに絞り、余白と読みやすさを優先しました。主ロゴやサブロゴ、文字色、光輪色はブラウザ上で調整できます。

## 使い方

`start-test-server.bat` を実行して `http://127.0.0.1:8000/index.html` を開く。入力欄を書き換え、ロゴや色を選び、プレビューを確認したら `PNG保存` または `SVG保存` を押す。GitHub Pages上ではそのまま `index.html` を開いて使える。

## 注意

これは非公式ファンメイドのサークル内利用を想定したデザイン案。公式ロゴ、公式イラスト、ゲーム内スクリーンショットを直接使用しない方針にしている。WebフォントはGoogle Fontsのオープンソースフォントを参照する。
